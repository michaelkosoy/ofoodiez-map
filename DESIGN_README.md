# Ofoodiez Design System

This document outlines the core design aesthetics and color palette used across the Ofoodiez website. Use this as a reference when building new components or pages to maintain visual consistency.

## Color Palette

Our website relies on a warm, organic palette inspired by food and nature. 

*   **Burgundy (Primary / Accent):** `#720815`
    *   Used for primary buttons, call-to-actions, calendar badges, and important highlights.
*   **Cream (Background):** `#f9f6eb`
    *   Used as the primary page background color. It provides a warm, soft canvas that is easier on the eyes than pure white.
*   **Darkest Brown (Primary Text):** `#362118`
    *   Used for primary headings and body text to maintain high contrast while feeling softer than pure black.
*   **Medium Brown (Secondary Elements):** `#603e35`
    *   Used for secondary text, borders, and subtle UI elements.
*   **Light Brown (Tertiary Accent):** `#845649`
    *   Used for decorative elements, hover states, and softer borders.

## Usage Guidelines

*   **Consistency:** Always use these exact hex codes. Avoid introducing new random colors without a specific reason.
*   **Contrast:** Ensure text (usually `#362118`) has sufficient contrast against its background.
*   **Aesthetics:** The design should feel premium, warm, and inviting. Use smooth gradients, rounded corners (where appropriate), and subtle shadows to enhance the UI.
